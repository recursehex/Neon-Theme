# Changelog

## [1.0.8] - 2026-06-08

### Changed

- Overhauled UI to use pure black (#000000/#0A0A0A) backgrounds instead of dark purple (#090A0F)
- Replaced all purple-tinted grays (#555B77, #3F445A, #35394B, #2C303F) with neutral grays (#676767, #878787, #A1A1A1, #1A1A1A, #1F1F1F, #2E2E2E)
- Softened foreground from pure white (#FFFFFF) to off-white (#EDEDED)
- Added visible borders (#2E2E2E) to sidebar, panel, activity bar, title bar, and tabs
- Python 'self' keyword now uses Blue
- Indigo color now correctly represents Constants instead of just Enums in the color palette
- Added extensive UI token coverage: breadcrumbs, checkboxes, debug tools, diff editor, notifications, symbol icons, minimap, welcome page, and more
- Updated README color swatches to use shields.io badges
